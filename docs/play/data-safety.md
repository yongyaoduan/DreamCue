# 数据安全填写建议

以下内容基于当前这版代码：

- 没有网络权限
- 备忘数据保存在本地 SQLite
- `allowBackup=false`
- 没有账号体系
- 没有分析埋点、广告 SDK、云同步、远程上传

## 建议答案

### 是否收集或共享任何用户数据

- 当前版本建议填写：`No`

原因：

- 备忘内容、时间、日志都只保存在用户设备本地
- 当前代码没有把这些数据发送到开发者服务器或第三方
- 没有广告、统计、云同步或账号登录链路

## 需要特别注意

- 如果你后面加入了云同步、崩溃上报、统计分析、登录系统、远程备份、AI 云搜索，这一页必须重新填写
- 如果你再次打开 Android 自动备份，也需要重新确认这页是否还应该填 “No”

## 本应用里实际存在的数据

- 备忘内容
- 添加时间
- 修改时间
- 消除时间
- 提醒状态与提醒次数

这些数据当前都只用于：

- 在本地展示
- 本地搜索排序
- 本地通知提醒

## 删除能力

- 用户可以消除某条备忘
- 用户也可以彻底删除某条备忘和相关日志
