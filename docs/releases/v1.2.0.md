# DreamCue v1.2.0

## Highlights

- Rebuilt the Android experience around the new DreamCue visual system.
- Added the Today, Recall, Archive, Rhythm, and Account navigation model.
- Added the calendar-style wheel picker for reminder time selection.
- Replaced the launcher icon with a design-derived adaptive icon and reused the same mark inside the Android UI.
- Added UI coverage for the redesigned navigation, capture flow, reminder picker, and brand details.

## Notes

- Firebase configuration is not committed. Fill Android string resources and `macos/DreamCueMac/FirebaseConfig.plist` before testing live sync.
- Android release assets include the signed APK. Play Store AAB delivery is handled separately.
- macOS behavior is unchanged from v1.1.0 in this release.
