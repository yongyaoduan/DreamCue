# DreamCue v1.2.7

## Highlights

- Keeps Archive cue details status-specific on macOS: active cues show Delete and Complete, while cleared cues show Delete and Return to Today.
- Clarifies Android reminder delivery with individual notifications for pinned due cues and one dismissible summary notification for regular due cues.
- Refines pinned cue row treatment on Android and macOS with a quieter neutral fill instead of a heavy visual marker.
- Updates Realtime Database rules to validate synced `display_order` and `pinned` fields.
- Keeps GitHub release assets limited to the Android APK and macOS zip.

## Verification

- Rust workspace tests pass.
- Android unit tests pass.
- Android connected UI tests cover Today ordering, pinned row state, Archive actions, Account copy, and reminder time selection.
- macOS XCUITest covers Archive-specific action sets, cue drag behavior, Account layout, and reminder time selection.
- Documentation checks pass for the edited release and architecture notes.
