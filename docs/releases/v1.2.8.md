# DreamCue v1.2.8

## Highlights

- Adds `macos/DreamCueMac/ReturnKeyCommitPolicy.swift` so the macOS editor checks `NSTextView.hasMarkedText()` before saving on Return.
- Removes the direct Return shortcut from the macOS Save buttons in `ContentView.swift`; Return now routes through the editor policy instead of bypassing input-method composition.
- Keeps Command Return as the line-break shortcut inside cue text.
